# 深入类型转换

## 前言

ES规范中定义了很多类型转换的函数，用于将任意的值转换为指定类型，其中重要的有以下几个：

- `ToBoolean` 转换为布尔类型
- `ToNumber` 转换为数值类型
- `ToString` 转换为字符串类型
- `ToPrimitive` 转换为基本数据类型

以上几种类型转换适用范围非常广，可以说我们无时无刻不在跟它们打交道：

```js
// ToBoolean
if (3) {} // true
if ([]) {} // true
if ('') {} // false

// ToNumber
const a = '3' - 1 // 2

// ToString
'1' + 'hello' // 1hello
[] + 'hello' // hello
{} + 'hello' // [object Object]hello

// ToPrimitive
[1] == '1' // true
[1] == 1 // true
[1] == true //  true
```
