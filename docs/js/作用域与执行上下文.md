# 作用域与执行上下文

## 作用域

JavaScript的作用域同大多数语言（如C、Java等）一样，都是 `词法作用域` ，词法作用域的典型特征是：**在书写代码时，就确定了作用域。**

```js
function foo() {
  var a = 1
  function bar() {
    console.log(a) // 1
  }
  bar()
}

foo()
```

上述代码一共有3个作用域：

1. 全局作用域
2. `foo` 函数作用域
3. `bar` 函数作用域

在编译阶段，变量 `a` 就确定存在于 `foo` 函数作用域中。

::: tip
编译阶段进行词法分析时，其实就是找到各作用域，并将作用域下声明的变量与当前作用域进行关联。
:::
