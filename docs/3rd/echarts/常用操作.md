## Tooltip

#### 自定义背景

```javascript
tooltip: {
  trigger: 'axis',
  axisPointer: {
    type: 'shadow',
    shadowStyle: {
      color: 'rgba(0, 0, 100, .1)'
    }
  }
}
```

## Legend 图例

#### 更改位置

```javascript
{
    legend: {
        // left/right/top/bottom
        bottom: 0 // 靠底部
    }
}
```

#### 更改大小

```javascript
{
    legend: {
        left: '20%',
        width: '60%'
    }
}
```
